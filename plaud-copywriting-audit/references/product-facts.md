## Common claim errors to flag

| Claim seen in copy | Correct fact | Action |
|---|---|---|
| "up to 16.4 ft" for any mode on Note Pro | 16.4 ft = Enhance mode only. Endurance = 9.8 ft. | Specify mode |
| "up to 50 hours" without qualifier | 50 hr = Endurance mode. Enhance = 30 hr. | Add mode |
| "4 MEMS mics" for Plaud Note | Plaud Note has 2 MEMS. 4 MEMS = Note Pro only. | Correct |
| "smart dual-mode" for Plaud Note | Smart (auto) = Note Pro. Manual = Plaud Note. | Correct |
| "SOC II" | Always "SOC 2 Type 2" | Correct |
| "unlimited storage" without qualifier | Unlimited = paid plans only | Add qualifier |
| "phone call recording" for NotePin / NotePin S | Neither NotePin supports calls. Note / Note Pro only. | Remove or correct |
| "joins your meeting" for Plaud Desktop | Plaud Desktop captures ambiently — it does NOT join as a bot | Correct |
| "10,000 templates" | Correct is "10,000+" | Add + |
| "mins/mo" | Correct is "min/mo" (no pluralizing abbreviations) | Correct |
